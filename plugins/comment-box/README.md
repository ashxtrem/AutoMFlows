# Comment Box Plugin

Add markdown comments to your workflow to document and explain your automation steps.

## Features

- **Markdown Support**: Write comments using markdown syntax for rich formatting
- **Visual Documentation**: Add comments near nodes to explain what they do
- **Resizable**: Adjust the size of comment boxes to fit your content
- **No Execution**: Comment boxes don't execute during workflow runs - they're purely visual

## Usage

1. Drag the "Comment Box" node from the Utility category in the sidebar
2. Click on the comment box to open the markdown editor
3. Write your comment using markdown syntax
4. Save to see the formatted comment displayed in the node

## Markdown Support

The comment box supports standard markdown syntax:
- Headers (`#`, `##`, `###`)
- Bold (`**text**`) and italic (`*text*`)
- Lists (ordered and unordered)
- Links (`[text](url)`)
- Code blocks and inline code
- And more!

## Example

```markdown
# Important Note
This workflow handles user authentication.

**Steps:**
1. Navigate to login page
2. Enter credentials
3. Verify success
```
